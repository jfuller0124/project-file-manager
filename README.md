# project-file-manager

A full-stack web application that allows users to upload, preview, lock, and manage project files securely in real time.

## Why I Built This
I built this project to solve issues I faced managing shared project files and preventing accidental overwrites.

## Features
- Secure file uploads and previews
- Real-time file locking
- Role-based access control using Supabase RLS
- Cloud storage integration

- ## Tech Stack
- Next.js
- TypeScript
- Supabase (PostgreSQL, Auth, Storage)
- Tailwind CSS

- ## What I Learned
- Designing secure backend workflows
- Implementing row-level security policies
- Building scalable, cloud-hosted systems
